Typography Poster

A poster design exploring how type, spacing, scale, and layout can create a strong visual style.

Objective

The goal of this project was to showcase a chosen font family and use typography as the main visual element. The poster focuses on hierarchy, contrast, and composition.

Result

The final design uses type as both information and image, giving the poster a clear focal point while keeping the layout readable.
